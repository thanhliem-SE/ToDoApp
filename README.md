# ToDoApp

## Description
A simple todo App for learning React-Native code app.

## What you'll need
* Android Emulator
* NodeJS
* NPM/NPX

## Run
```
git clone https://github.com/thanhliem-SE/ToDoApp.git
```
```
cd ToDoApp
```
```
npm install
```
```
npx react-native start
```
```
npx react-native run-android
```
